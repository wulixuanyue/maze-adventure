# 🎮 迷宫冒险 - Maze Adventure

> 一款基于 HTML5 Canvas 的网页迷宫冒险游戏，为《项目实训（一）》课程开发。

🔗 **在线试玩**：[https://wulixuanyue.github.io/maze-adventure](https://wulixuanyue.github.io/maze-adventure)

---

## 📖 项目介绍

迷宫冒险是一款经典的迷宫解谜游戏。玩家需要控制角色在随机生成的迷宫中寻找出口，收集星星获取高分，挑战越来越大的迷宫关卡。

### 核心玩法
- 使用 **WASD** 或 **方向键** 控制角色移动
- 找到出口 🏁 即可过关
- 收集星星 ⭐ 获得额外分数
- 步数越少、时间越短，得分越高

---

## ✨ 功能特性

### 游戏系统
- ✅ **随机迷宫生成** — 基于递归回溯算法，每次关卡都是全新迷宫
- ✅ **5 个递进关卡** — 迷宫尺寸逐渐增大，难度递增
- ✅ **计时与计步** — 实时显示通关时间和移动步数
- ✅ **暂停/继续** — 支持游戏中随时暂停
- ✅ **重新开始** — 可重开本关或返回菜单

### 排行榜系统
- ✅ 记录玩家昵称、关卡、用时、步数、星星数、得分
- ✅ 支持按 **最高得分** 和 **最快通关** 排序
- ✅ 本地持久化存储（localStorage）
- ✅ 支持清空排行榜记录

### 成就系统（6个成就）
| 成就 | 描述 |
|------|------|
| 🏁 初次通关 | 完成第一关 |
| ⚡ 速度之王 | 60秒内完成一关 |
| ⭐ 星星收集者 | 单关收集全部星星 |
| 👣 迷宫马拉松 | 累计移动500步 |
| 👑 迷宫大师 | 通关全部5关 |
| 🎯 完美挑战 | 不收集星星直接通关 |

### 分享功能
- ✅ 生成成绩分享文本（可复制到剪贴板）

### 视觉与体验
- ✅ **加载画面** — 游戏启动时显示加载动画
- ✅ **封面菜单** — 精美的游戏封面与导航菜单
- ✅ **操作引导** — 首次进入游戏显示操作说明（可勾选不再显示）
- ✅ **过关动画** — 关卡完成时显示统计数据与解锁成就
- ✅ **深色主题** — 统一的深色游戏风格
- ✅ **响应式布局** — 适配桌面端与移动端

### 个人简历页面
- ✅ 从游戏封面、暂停菜单均可跳转至「关于我」页面
- ✅ 包含完整的个人信息、技能标签、项目经历、联系方式
- ✅ 风格与游戏主体统一

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 Canvas | 游戏画面渲染 |
| 原生 JavaScript (ES6+) | 游戏逻辑与交互 |
| CSS3 | 界面样式与动画 |
| LocalStorage | 本地数据持久化 |
| GitHub Pages | 项目部署 |

### 性能优化
- 使用 `requestAnimationFrame` 实现流畅渲染循环
- 使用 `Uint8Array` 存储迷宫数据，节省内存
- 局部重绘机制，减少不必要的渲染开销
- 离屏计算迷宫生成，避免阻塞主线程
